💡 Core Technology Stack
========================

- **Backend:** Python 3.11+ using `py-cord` and `ezcord`.
- **API:** FastAPI with Uvicorn.
- **Frontend:** React + TypeScript + Vite + Tailwind CSS.
- **Database:** MariaDB (Recommended) or SQLite.
